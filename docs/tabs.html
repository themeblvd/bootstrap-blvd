Deep Linking Tabs

Tab deep linking allows you to open a specific tab on page load by passing in the cooresponding hashtag in the URL. For this to work you need to add class tabs-deep-link to your .fs-tabs element. And then when you link to your page containing the tabbed content, you'll add a hash to your URL formatted like #tab_{id of tab to open}.

For example, the following set of tabs has three content panes with the ID's, alpha, beta, and theta. If we wanted to open the beta pane on page load, we would link to tabs.html#tab_beta.
