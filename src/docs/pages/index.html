Core

Below are the core elements of Front Street. The core package includes styling for only what you see below and requires no JavaScript. If you'd like to use Front Street's core by itself, we've compiled a frontstreet-core.css file for you.

  • Buttons Many different practical types of buttons.
  • Forms Field types and sizes, and form layouts.
  • Grid Flexbox grid system, 10-column and 12-column.
  • Lists Various ways to use HTML lists.
  • Media Responsive images, videos and galleries.
  • Quotes Blockquotes in various layouts and formats.
  • Tables All the default styling for responsive tables.
  • Typography Various headings and other useful typographical elements.
  • Utilities Useful utilities for displaying, alignment, etc.

Blocks

Below are all the blocks included with Front Street, which extend the core elements above. If you're looking to implement Front Street in its entirety, you'll simply include the frontstreet.css and frontstreet.js files in your website.

  • Alerts Contextual display notices.
  • Backgrounds Background shading, parallax and sliders.
  • Breadcrumbs Classic navigational necessity.
  • Dividers Horizontal rules and underlines.
  • Loader Animated multi-purpose loader.
  • Models Custom integration of Magnific Popup.
  • Navigation Various responsive navigational elements.
  • Nav Mobile Mobile-only navigation.
  • Progress Bars Display progress of tasks as a percentage.
  • Sliders Custom integration of Owl Carousel.
  • Tabs Organize content with a tabbed navigation.
  • Testimonials Quotes to highlight client feedback.
  • Toggles Show, hide and group content.
  • Tooltips Classic tooltips for hovered links.
  • Widgets Containers for aside content.

Extras

This section includes a few extra features for you. — The color library is compiled as a separate frontstreet-colors.css file, you'd include separately (this probably isn't something most projects would be using). And the helpers come with the default implementation of frontstreet.css, but will not be included if you use frontstreet-core.css.

  • Color Library Predefined library of color classes.
  • Helpers Helper classes that aren't blocks and aren't part of core.
